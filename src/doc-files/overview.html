JJCronCommon is library for JJCron cron-like daemon and JJCronRM, remote manager of JJCron instances. JJCronCommon contains two packages one with shared functionality of them both and the other one with interfaces or classes explicitly used for remote communication.

More Information

Further information can be found on official JJCron Github organization page.
